Detailed Description

This feature computes the hash of an object

  • If a the call hash_func(i_object) is legal, it is used to compute the hash. This function should be defined in the namespace that contains TYPE (it will use ADL). If such function exits, its return type must be size_t.
  • Otherwise std::hash<TYPE> is used to compute the hash
