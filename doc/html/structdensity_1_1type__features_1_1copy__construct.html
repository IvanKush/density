Detailed Description

This feature stores a pointer to the copy constructor in the table of the type. The effect of the feature is copy-constructing an object of type TYPE, and returning a pointer (of type BASE) to it.

Parameters
i_complete_destpointer to the storage in which the TYPE must be constructed.
i_base_sourcepointer to a subobject (of type BASE) of an object whose complete type is TYPE, that is to be used as source of the copy.
Returns
pointer to a subobject (of type BASE) of the new object.
